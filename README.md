# Vocaloid Essentials - Recommended Songs Showcase

A beautifully designed, premium interactive web portal curating iconic, legendary, and trending Vocaloid songs, optimized for English-speaking fans and newcomers.

👉 Live Demo: [Deploy to GitHub Pages to get your link]

## Features

- **Curated Catalog**: Features 12 history-making songs across different Eras (Classic to Modern) featuring diverse Vocalists (Hatsune Miku, Kagamine Rin/Len, Megurine Luka, Megpoid Gumi, IA, and v flower).
- **Responsive Cyberpunk Aesthetic**: Modern design featuring dark-cyber space theme, neon glow indicators custom-fitted to character themes, and premium typography (`Outfit` & `Inter`).
- **Interactive Multi-Filter System**: Filter the catalog in real-time by singer, musical vibe (Pop, Rock, Classic, EDM, Dark, Emotional, Energetic), or search by title, Japanese title, producer, or description.
- **Embedded Audio/Video Player Modals**: Listen directly using non-intrusive modal windows containing official YouTube MV streams with zero page reloads.
- **Educational Guide**: An introductory section helping English-speaking audiences understand the technology, cooperative culture, and global popularity of Vocaloid.

## Technology Stack

- **Markup & Structure**: HTML5 Semantic elements
- **Styling**: Vanilla CSS3 (Custom property variables, glassmorphic filters, keyframe animations, CSS grid, flex layouts)
- **Logic**: Vanilla ES6 JavaScript (No frameworks needed, ensuring lightning-fast load times and seamless GitHub Pages compatibility)

## Quick Start (Run Locally)

You do not need to install any heavy packages or compilers.

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/vocaloid-essentials.git
   cd vocaloid-essentials
   ```
2. **Open index.html**:
   - Double-click `index.html` to open it in your browser.
   - *Recommended*: Run it with a local development server (like VS Code Live Server extension) for optimal iframe performance.

## Hosting on GitHub Pages

This project is 100% static and optimized for zero-config GitHub hosting:

1. Push this folder to your public GitHub repository.
2. In the repository settings, go to the **Pages** tab.
3. Under **Build and deployment**, select **Deploy from a branch**.
4. Choose the `main` (or `master`) branch and `/root` directory, then click **Save**.
5. Your website will be live at `https://<your-github-username>.github.io/<repository-name>/` in a few minutes!
